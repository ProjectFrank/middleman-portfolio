Why I got into web development

Sep 7 2014

This is a blog entry is incomplete and will be updated.

After finishing my undergrad last year, I never thought I’d be going back to school again. What’s more, I’m actually really excited about going back to school. I’ve enrolled in the Front-End Bootcamp at HackerYou and I start next Monday.

When I graduated from my program, I had the opportunity to start my own guesthouse business (B&B style accommodation minus breakfast). In order to have a presence on the internet, I had to make myself a website. My brother, who is a full-stack web developer, offered to help me put together a site using a Bootstrap template and ComfortableMexicanSofa. Unfortunately, the site was never completed. Between managing all the other aspects of the business, taking pictures, and writing content for the site, it became apparent that I had bitten off way more than I could chew. However, I discovered that web development was something I loved. I got myself out of the business this summer and I have been learning about Front-End Web Development every day since.

Prior to this summer, I have had minimal experience with coding besides working with java in highschool and building geocities sites as a kid. I've been using some amazing online resources to accelerate my learning over the past couple of months that I'd recommend to anyone who also wants to get started along this path:

  • CS50x
  • Eloquent Javascript
  • Hack Design
  • Codecademy
  • Git Immersion
  • Level Up Tuts (specifically the CSS Experiments Series)

I’m really happy with the progress I’ve made so far. I’ve learned a fair bit about HTML5/CSS3/Sass, Git, JavaScript and jQuery. I’ve also become familiar with really awesome tools such as Emacs, Middleman, and Sublime Text.
